\doxysubsection{Detailed Description}
Vulkan resource creation functions for the renderer subsystem. 

All functions follow the same convention\+: inputs are const refs, outputs are raw pointers prefixed with p. On failure, returns a Failure result with a location-\/prefixed message. On success, returns \doxylink{clz__types_8hpp_ab9f1aa62b911366b18aeedcfc204901d}{clz\+::\+Success()}. None of these functions clean up on partial failure — callers are responsible for cleanup if INVALIDATE triggers a bail-\/out.

Creates the graphics pipeline and pipeline layout. 

Hardcoded for a no-\/vertex-\/buffer triangle pipeline\+:
\begin{DoxyItemize}

\item Triangle list topology
\item Back-\/face culling, clockwise winding

\item No blending
\item Dynamic rendering (no render pass object — uses Vk\+Pipeline\+Rendering\+Create\+Info)
