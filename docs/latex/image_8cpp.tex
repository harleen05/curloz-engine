Inserts a Vk\+Image\+Memory\+Barrier2 to transition an image\textquotesingle{}s layout. 

Inserts a pipeline barrier to transition an image\textquotesingle{}s layout.

Builds a Vk\+Image\+Memory\+Barrier2 targeting a single mip level and single array layer of a color image, wraps it in a Vk\+Dependency\+Info, and records it with vk\+Cmd\+Pipeline\+Barrier2 (sync2 / Vulkan 1.\+3).

Queue family ownership is not transferred (VK\+\_\+\+QUEUE\+\_\+\+FAMILY\+\_\+\+IGNORED).
