\doxysubsection{Detailed Description}
Vulkan object wrapper types for the Curloz Engine renderer. 

Each struct is a thin named wrapper around one or more raw Vulkan handles. All handles are default-\/constructed to VK\+\_\+\+NULL\+\_\+\+HANDLE so uninitialized state is detectable. Owned and populated by \doxylink{initializers_8hpp}{initializers.\+hpp}; destroyed by the matching functions in \doxylink{cleaners_8hpp}{cleaners.\+hpp}.
