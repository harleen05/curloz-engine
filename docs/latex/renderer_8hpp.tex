\doxysubsection{Detailed Description}
Public interface for the Curloz Engine renderer subsystem. 

One-\/stop include for anything render-\/related. Mirrors the window subsystem pattern — callers never touch Vulkan handles directly. All module-\/level state and Vulkan objects live in \doxylink{renderer_8cpp}{renderer.\+cpp}.

Typical call order\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{render::init();}
\DoxyCodeLine{\textcolor{keywordflow}{while}\ (!window::windowShouldClose())}
\DoxyCodeLine{\ \ \ \ render::update(dt);}
\DoxyCodeLine{render::cleanup();}

\end{DoxyCode}

Brings up the full Vulkan renderer stack. 

Initializes the full Vulkan renderer stack.

Initializes all subsystems in dependency order. Any failure propagates immediately via INVALIDATE, leaving partial state uncleaned (full init is a one-\/time path; partial cleanup is not implemented).

Shader paths are hardcoded to\+: assets/shaders/triangle\+\_\+vert.\+spirv assets/shaders/triangle\+\_\+frag.\+spirv

\begin{DoxyReturn}{Returns}
Success if all subsystems came up, Failure otherwise.
\end{DoxyReturn}
Brings up subsystems in order\+:
\begin{DoxyEnumerate}
\item Vulkan instance + validation layers + surface + device + queues
\item Swapchain
\item Shader modules + graphics pipeline
\item Sync primitives (semaphores + fences)
\item Command pool + command buffers
\end{DoxyEnumerate}

Uses INVALIDATE throughout — any failure propagates immediately.

\begin{DoxyReturn}{Returns}
Success if all subsystems initialized, Failure with a message otherwise. 
\end{DoxyReturn}

Renders one frame using the double-\/buffered frame-\/in-\/flight loop. 

All steps are indexed by r\+\_\+current\+Frame except r\+\_\+image\+Index which comes from vk\+Acquire\+Next\+Image\+KHR and may differ from r\+\_\+current\+Frame when the swapchain has more images than FRAMES\+\_\+\+IN\+\_\+\+FLIGHT.

r\+\_\+current\+Frame is advanced at the end with a modulo wrap.

Per-\/frame sequence\+:
\begin{DoxyEnumerate}
\item Wait for the in-\/flight fence (CPU-\/\+GPU sync)
\item Acquire next swapchain image
\item Reset + begin command buffer
\item Record draw commands (image transitions + dynamic rendering + draw call)
\item Submit command buffer to graphics queue
\item Present to swapchain
\item Advance r\+\_\+current\+Frame
\end{DoxyEnumerate}
