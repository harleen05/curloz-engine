\doxysubsection{Detailed Description}
Core result type and error-\/propagation utilities for Curloz Engine. 

Defines \doxylink{structclz_1_1Result}{clz\+::\+Result}, factory helpers (Success / Failure), and the CHECK / INVALIDATE macros that form the engine\textquotesingle{}s lightweight error handling convention — no exceptions, no std\+::expected, just a bool and a message.
