\doxysubsection{Detailed Description}
Internal Vulkan-\/surface interface for the window subsystem. 

Low-\/level functions that bridge GLFW and Vulkan, operating directly on a types\+::\+Window handle. Not part of the public engine API — use \doxylink{window_8hpp}{window.\+hpp} instead, which wraps these with the module-\/level instance.

Queries the Vulkan instance extensions required by GLFW. 

Calls glfw\+Get\+Required\+Instance\+Extensions() to get the platform surface extensions (e.\+g. VK\+\_\+\+KHR\+\_\+surface + VK\+\_\+\+KHR\+\_\+xcb\+\_\+surface on Linux/\+X11, VK\+\_\+\+KHR\+\_\+wayland\+\_\+surface on Wayland) and appends each name pointer to {\ttfamily Extensions}.

\begin{DoxyNote}{Note}
