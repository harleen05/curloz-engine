\doxysubsection{Detailed Description}
Stateless public forwarders and module-\/level window state. 

Owns the single module-\/level types\+::\+Window instance (w\+\_\+window) and the default dimensions (w\+\_\+width, w\+\_\+height). Every function here is a thin forwarder that injects w\+\_\+window into the real implementation living in \doxylink{native_8cpp}{native.\+cpp} or \doxylink{render_8cpp}{render.\+cpp}.

Callers never see w\+\_\+window — they go through \doxylink{window_8hpp}{window.\+hpp}, which exposes the same signatures without the window parameter.
