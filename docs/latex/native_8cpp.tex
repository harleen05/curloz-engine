\doxysubsection{Detailed Description}
GLFW lifecycle implementation for the window subsystem. 

Real implementations of init / update / cleanup / window\+Should\+Close. All functions take an explicit types\+::\+Window\texorpdfstring{$\ast$}{*} so they carry no global state themselves — the module-\/level w\+\_\+window is owned by \doxylink{window_8cpp}{window.\+cpp} and injected here via the forwarding layer.
