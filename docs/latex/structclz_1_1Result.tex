\doxysubsection{Detailed Description}
Represents the outcome of a fallible engine operation. 

Carries a boolean success flag and an optional human-\/readable message. The default constructor produces a successful result. Use the \doxylink{clz__types_8hpp_ab9f1aa62b911366b18aeedcfc204901d}{Success()} and \doxylink{clz__types_8hpp_ad95e747a3a7b5370d8e85aab198ac4b9}{Failure()} factory functions rather than constructing \doxylink{structclz_1_1Result}{Result} directly.
