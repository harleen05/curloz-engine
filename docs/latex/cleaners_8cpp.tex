\doxysubsection{Detailed Description}
Vulkan resource destruction implementations. 

Each function is the exact counterpart of a creation function in \doxylink{initializers_8hpp}{initializers.\+hpp} or \doxylink{commandbuffer_8hpp}{commandbuffer.\+hpp}. Call order in \doxylink{renderer_8cpp}{renderer.\+cpp} follows strict reverse-\/init order to avoid use-\/after-\/free of dependent handles.
