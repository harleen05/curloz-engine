\doxysubsection{Detailed Description}
Holds all per-\/frame synchronization primitives. 

All three vectors are sized to FRAMES\+\_\+\+IN\+\_\+\+FLIGHT.
\begin{DoxyItemize}
\item image\+Available\+Semaphores\+: signaled when a swapchain image is ready to render into.
\item render\+Complete\+Semaphores\+: signaled when rendering is done, waited on by present.
\item in\+Flight\+Fence\+: CPU-\/\+GPU sync; waited on at the start of each frame to avoid overwriting resources still in use by the GPU. 
\end{DoxyItemize}
