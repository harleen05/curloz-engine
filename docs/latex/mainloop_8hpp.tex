\doxysubsection{Detailed Description}
Per-\/frame Vulkan render loop operations for the renderer subsystem. 

These functions implement the six steps of the double-\/buffered render loop. They are called in sequence by render\+::update() in \doxylink{renderer_8cpp}{renderer.\+cpp} — not meant to be called directly by external code.

Call order per frame\+: wait\+For\+GPU → acquire\+Next\+Image → start\+Command\+Buffer → record\+Command\+Buffer → submit\+Command\+Buffer → present
