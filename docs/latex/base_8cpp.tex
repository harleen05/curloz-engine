\doxysubsection{Detailed Description}
Implementation of the core engine lifecycle functions. 

Drives initialization, the main loop, and shutdown for all internal subsystems (window and renderer). Entry point for engine bring-\/up is \doxylink{base_8cpp_ad23afb2b76e187be63025ab887353782}{clz\+::init()}, followed by \doxylink{base_8cpp_a10f36af73a4ed3a2382c47f1ef2deccb}{clz\+::update()}, then \doxylink{base_8cpp_a296c96a54e9ca399433acd6d0be55e75}{clz\+::cleanup()}.

Initializes all internal engine subsystems.

Calls window\+::init() first, then render\+::init(). Uses the CHECK macro to bail out early if either step fails. Prints confirmation to stdout on success.

Runs the engine\textquotesingle{}s main loop. 

Runs the main engine loop until the window signals a close.

Pumps window events and renderer updates every frame. Runs until window\+::window\+Should\+Close() signals that the window has been closed or a quit event was received.

\begin{DoxyNote}{Note}
The delta-\/time argument passed to window\+::update() and render\+::update() is currently hardcoded to 0 (placeholder).
\end{DoxyNote}
