Inserts a VkImageMemoryBarrier2 to transition an image's layout.

Inserts a pipeline barrier to transition an image's layout.

Builds a VkImageMemoryBarrier2 targeting a single mip level and single array layer of a color image, wraps it in a VkDependencyInfo, and records it with vkCmdPipelineBarrier2 (sync2 / Vulkan 1.3).

Queue family ownership is not transferred (VK_QUEUE_FAMILY_IGNORED).

Parameters
imageImage to transition.
oldLayoutLayout the image is currently in.
newLayoutLayout to transition to.
src_access_maskMemory access type that was performed before this barrier.
dst_access_maskMemory access type that will be performed after this barrier.
src_stage_maskPipeline stage that produced the previous access.
dst_stage_maskPipeline stage that will perform the next access.
commandBufferCommand buffer to record the barrier into.

Wraps VkImageMemoryBarrier2 + VkDependencyInfo + vkCmdPipelineBarrier2 (Vulkan 1.3 / synchronization2). Transitions a single-mip, single-layer color image. Queue family ownership is not transferred (VK_QUEUE_FAMILY_IGNORED on both sides).

Common usage in the render loop:

  • UNDEFINED → COLOR_ATTACHMENT_OPTIMAL before rendering
  • COLOR_ATTACHMENT_OPTIMAL → PRESENT_SRC_KHR before presentation
