Detailed Description

Internal GLFW lifecycle interface for the window subsystem.

Low-level functions that operate directly on a types::Window handle. Not part of the public engine API — use window.hpp instead, which manages the module-level Window instance and forwards here.

◆ init()

clz::Result clz::window::init ( types::Window * pWindow)

Initializes GLFW and creates a Vulkan-compatible window.

Initializes GLFW and creates a window into pWindow.

Sequence:

  1. glfwInit() — returns Failure on error.
  2. Set GLFW_CLIENT_API to GLFW_NO_API (Vulkan, no GL context).
  3. Set GLFW_RESIZABLE to false (fixed 800×600 for now).
  4. glfwCreateWindow() — returns Failure if handle is null.
Parameters
pWindowNon-null pointer to the Window struct to populate.
Returns
Success with a log message, or Failure with an error location string.

Calls glfwInit(), applies Vulkan/no-resize hints, and creates an 800×600 window. On any failure returns a Failure result with a descriptive message.

Parameters
pWindowNon-null pointer to the Window struct to populate.
Returns
Success with a log message, or Failure on GLFW errors.
