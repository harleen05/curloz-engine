Detailed Description

Vulkan-surface and extension-query implementation for the window subsystem.

Bridges GLFW and Vulkan. Like native.cpp, these functions take an explicit types::Window reference and carry no global state — w_window is injected by the forwarding layer in window.cpp.

Creates a Vulkan surface backed by the GLFW window.

Wraps glfwCreateWindowSurface(). The caller owns the returned surface and must destroy it with vkDestroySurfaceKHR() before destroying the VkInstance.

Parameters

Queries the Vulkan instance extensions required by GLFW.

Calls glfwGetRequiredInstanceExtensions() to get the platform surface extensions (e.g. VK_KHR_surface + VK_KHR_xcb_surface on Linux/X11, VK_KHR_wayland_surface on Wayland) and appends each name pointer to Extensions.

Note
The returned pointers are owned by GLFW and remain valid until glfwTerminate() is called. Do not free them.
Parameters
ExtensionsOutput vector to append extension name strings to.
extensionsCountOutput count populated by GLFW.
windowThe active Window (establishes GLFW context).
